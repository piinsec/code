သင်ခန်းစာ ၁ - Password Cracking အခြေခံများ

Password cracking ဆိုတာက အားနည်းတဲ့ password တွေကို ခန့်မှန်းဖောက်ထွင်းနည်းပါ။ ethical hacking မှာ system ရဲ့ အားနည်းချက်ကို စမ်းသပ်ဖို့အတွက် အသုံးပြုပါတယ်။

အသုံးများတဲ့ password cracking နည်းလမ်းများ:

  • Brute Force Attack - ဖြစ်နိုင်သမျှ combination အားလုံးကို စမ်းသပ်ခြင်း
  • Dictionary Attack - အသုံးများတဲ့ password list သုံးပြီး စမ်းသပ်ခြင်း
  • Rainbow Table Attack - hash ကနေ original password ကို ပြန်ရှာခြင်း
သတိပြုရန်: ဒီနည်းပညာတွေကို သင့်ကိုယ်ပိုင် system ဒါမှမဟုတ် permission ရထားတဲ့ system မှာပဲ စမ်းသပ်ပါ။ တခြားသူတွေရဲ့ system ကို ခွင့်ပြုချက်မရှိဘဲ စမ်းသပ်ရင် တရားဥပဒေအရ အရေးယူခံရနိုင်ပါတယ်။

John the Ripper သုံးပြီး password crack လုပ်နည်း (နမူနာ):

# John the Ripper ကို install လုပ်ပါ
sudo apt install john

# Password file ကို crack လုပ်ပါ
john --wordlist=/usr/share/wordlists/rockyou.txt hashed_passwords.txt

သင်ခန်းစာ ၂ - SQL Injection အခြေခံများ

SQL Injection ဆိုတာ website တစ်ခုရဲ့ database ကို ထိပါးနိုင်တဲ့ အခြေခံကျတဲ့ web application အားနည်းချက်တစ်ခုပါ။

SQL Injection အမျိုးအစားများ:

  • Error-based SQLi - Database error messages ကနေ information ရယူခြင်း
  • Union-based SQLi - UNION SQL operator သုံးပြီး data ရယူခြင်း
  • Blind SQLi - Error မပြဘဲ true/false response ကို စစ်ဆေးခြင်း

နမူနာ SQL Injection:

-- ရိုးရိုး login query
SELECT * FROM users WHERE username='admin' AND password='12345'

-- SQL Injection သုံးပြီး bypass လုပ်နည်း
username: admin' --
password: [anything]

-- Resulting query:
SELECT * FROM users WHERE username='admin' --' AND password='[anything]'
ကာကွယ်နည်း: Prepared statements သုံးပြီး SQL Injection ကို ကာကွယ်နိုင်ပါတယ်။

သင်ခန်းစာ ၃ - Network Scanning with Nmap

Nmap (Network Mapper) ဆိုတာ network discovery နဲ့ security auditing အတွက် အသုံးဝင်တဲ့ tool တစ်ခုပါ။

Nmap အခြေခံ commands:

  • Host Discovery - Network ပေါ်က live hosts တွေကို ရှာဖွေခြင်း
  • Port Scanning - Open ports တွေကို စစ်ဆေးခြင်း
  • Service Detection - Running services တွေကို ဖော်ထုတ်ခြင်း
  • OS Detection - Operating system ကို ခန့်မှန်းခြင်း

Nmap နမူနာ commands:

# Basic scan
nmap 192.168.1.1

# Scan specific ports
nmap -p 80,443 192.168.1.1

# Aggressive scan (OS and service detection)
nmap -A 192.168.1.1

# Scan entire subnet
nmap 192.168.1.0/24
သတိပြုရန်: သင့်ကိုယ်ပိုင် network မှာပဲ scan လုပ်ပါ။ တခြားသူတွေရဲ့ network ကို ခွင့်ပြုချက်မရှိဘဲ scan လုပ်ရင် ဥပဒေအရ အရေးယူခံရနိုင်ပါတယ်။
